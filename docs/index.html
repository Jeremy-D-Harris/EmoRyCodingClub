Pre-work questions

If you you have no previous experience with R or coding, then before arriving please complete the pre-work below to get familiar with a few basic things about the R Studio IDE and coding in R. The exercise below should take less than 5 minutes to complete.

Once you have installed R and R Studio, copy and paste the code below into the R Studio Console and press Enter to run it.

# Caculate something simple

Use the assignment operator, <-, to create new objects in R

c <- "Hello World"

Once we’ve completed the R for Data Science exercises, we’ll repeat the plotting exercise using open-access data from AirBnb.

This is a really big data set (it has ~50k rows !!), so it will take a moment to run. To import the AirBnb data from the URL, copy and paste the lines of code below:

require(readr)

# function for reading in zipped file (106 cols)
url <- "http://data.insideairbnb.com/united-states/ny/new-york-city/2019-06-02/data/listings.csv.gz"

url_read <- function(file_url) {
    con <- gzcon(url(file_url))
    txt <- readLines(con)
    return(read.csv(textConnection(txt)))
}

df <- url_read(url)  # read in data

# smaller csv file (16 cols)
url <- "http://data.insideairbnb.com/united-states/ny/new-york-city/2019-06-02/visualisations/listings.csv"
df <- read.csv(url)
